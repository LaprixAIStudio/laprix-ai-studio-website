Laprix Chat Admin belépés

Ez a felület csak belső használatra készült. A chat adatok, emberi átvétel, törlés, státuszok és belső jegyzetek csak érvényes admin tokennel érhetők el.

Token nélkül az admin API nem ad vissza adatot. Alapból nincs automatikus belépés; csak akkor jegyzi meg a tokent, ha bepipálod. Ezt az oldalt a keresők számára noindex jelöléssel látjuk el.
